NormJam

A Normalization Workshop and Jamboree

When and Where

NormJam is a 2 day workshop on the topic of normalization of single cell data hosted by the Chan Zuckerberg Initiative and the New York Genome Center.

The workshop will be held on November 19th and 20th, 2019 at the New York Genome Center, 101 6th Ave, New York, NY 10013.

About NormJam

Normalization refers to the pre-processing of a single expression matrix from a scRNA-seq dataset, in isolate, to remove technical variation in read counts across cells and molecules/genes, while maintaining biological variability.

What's the problem?

There are a wide variety of algorithms that exist for normalization, aka within sample removal of technical variability. Each of these algorithms result in differing outputs, and any error from normalization propagates to all subsequent analysis steps. So it is really important to normalize properly!

However, it is hard to evaluate which normalization algorithm to use and when. Many computational biologists working in single cell biology have been focused on downstream analysis, like batch effect correction (across sample removal of technical variability), data integration (within a modality, across experiments, assays, etc.), and multi-modal data integration (e.g., integrating smFISH with scRNA-seq data). We want to spend a few days thinking deeply about this problem and doing some initial benchmarking of methods.

What are our goals?

Bring together experts working on normalization to discuss and ideate

Define normalization, together

Apply existing normalization algorithms to a common set of benchmark data from different tissues and technologies to compare and contrast performance

Communicate our findings to the broader community

What don't we know?

How do we precisely define normalization?

How do we define what success looks like for a normalization algorithm? What metrics should we use?

What data should we use for benchmarking? How many tissues? How many modalities?
